# Security Audit Report

This file tracks known security findings and their resolution status for the Whale Watcher standalone repo.

## Status

No open findings.

## How to Add a Finding

When a security issue is identified and fixed, document it here:

```
### [Date] — [Short title]
- **Severity:** Low / Medium / High / Critical
- **Finding:** Description of the issue
- **Resolution:** What was changed and in which commit/PR
- **Verified by:** Who confirmed the fix
```

## Reporting New Issues

See [SECURITY.md](SECURITY.md) for the responsible disclosure process. Do not document unresolved vulnerabilities in this file.
